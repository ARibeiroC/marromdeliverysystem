# Marrom Delivery System
Sistema desenvolvido para gerir o fluxo de saídas de entregas da empresa durante o més, nele temos um fluxo de caixa simples que contabiliza e mostra o total de saidas e a quantidade a receber dentro de um intervalo de periodos, utilizado com filtro.

## STACK
Para este projeto foi utilizar React com Vite e javascript, para estilização eu utilizei uma abordagem modular, onde para cada componente foi criado um estilo arquivo de estilos, seguindo quase que o padrão antigo onde o CSS era emplementado diretamente no arquivo .jsx, assim facilitando a estilização e manutenção.
